# Sat-n-Alma-Uygulama

## Description
This is a basic C# Windows Forms application designed to calculate the total price of products entered by the user. It allows users to add product names and prices, display them in a list, and finalize the payment.

## Features
- Add product name and price
- Display added products in a list
- Calculate and show total price
- Clear all entries after payment

## Installation
1. Ensure you have Visual Studio installed on your machine.
2. Clone this repository to your local machine.
3. Open the project in Visual Studio.
4. Build and run the application.

## Usage
1. Enter a product name in the text box labeled 'Ürün Adı'.
2. Enter the price of the product in the text box labeled 'Fiyat'.
3. Click the 'Ekle' button to add the product to the list.
4. Repeat steps 1-3 for each product.
5. Click the 'Ödeme Tamamla' button to finalize the payment and clear all entries.
